# Viral Formulas

Use these formulas to generate original workplace copy. They are patterns, not fixed lines.

## Article Skeleton

Default article length: 10-12 image-ready lines, organized as 5-6 paired image groups.

Use this format:

- `第1组 上：...`
- `第1组 下：...`

Each pair should work as two related images:
- Upper line: scene, problem, or small feeling.
- Lower line: turn, relief, self-comfort, or deadpan punchline.

Default group progression:

1. Work positioning: work is livelihood, not life.
2. Relationship downgrade: coworkers/leaders/company are work relationships.
3. Money or energy anchor: salary does not deserve emotional overinvestment.
4. Specific scene: instant replies, meetings, overtime, weekly reports, team building, blame.
5. Emotional stop-loss: keep energy, mood, and boundaries.
6. Final comfort: give the reader a reason to feel less guilty.

Do not make all 10-12 lines equally sharp. The article should feel like continuous self-comfort with a few memorable lines.

## Paired Image Formulas

- Setup + turn:
  - 上：`下班后看到工作消息，心还是会抖一下`
  - 下：`后来才发现，慢一点回复也不会塌天`
- Problem + comfort:
  - 上：`有些事在群里看起来很急`
  - 下：`但真急的事，一般不会只发一句在吗`
- Workplace scene + self-protection:
  - 上：`工资是按月发的`
  - 下：`情绪就别按秒交付了`
- Relationship downgrade:
  - 上：`同事聊得来是运气`
  - 下：`聊不来也不用硬凑人生`
- Absurd comparison + release:
  - 上：`有些会开得像公司明天要上市`
  - 下：`但我只是想准点吃上晚饭`

## High-Resonance Motifs

### 工资不配消耗我

Core emotion: self-protection.

Good angles:
- 秒回焦虑
- 下班消息
- 周末焦虑
- 替公司操心
- 为同事破防
- 被领导一句话影响一天

Pattern:
- `这点工资不配让我秒回`
- `钱没到位，心别先到岗`
- `工作可以认真，情绪别白送`

### 上班比要饭难

Core emotion: absurd comparison.

Good angles:
- 要饭不用开会
- 要饭不用写周报
- 要饭不用团建
- 要饭不用背锅
- 要饭至少现结

Pattern:
- `要饭只是低头，上班还要汇报`
- `要饭不用复盘，上班连崩溃都要闭环`

### 同事只是同事

Core emotion: relationship downgrade.

Good angles:
- 不必深交
- 不必强行维系
- 不必替别人兜底
- 共事不是共命
- 聊得来是运气，聊不来也正常

Pattern:
- `同事只是共事，不是共生`
- `能对齐工作就够了，别对齐人生`

### 工作只是谋生

Core emotion: emotional stop-loss.

Good angles:
- 别卖命
- 别把公司当家
- 别把评价当判决
- 别让工作消耗最贵的自己

Pattern:
- `上班只是谋生，不是把自己交出去`
- `工作消耗不了你，才算你赢了`

## Anti-Fatigue Rotation

If a recent post used a salary anchor, the next post should use:
- coworker boundaries
- leader absurdity
- meeting nonsense
- message anxiety
- performance review

If a recent post used `这点工资不配`, avoid using it as the first line again. Use another surface:
- `钱没到位，心别先到岗`
- `这份工作，配不上我下班后的焦虑`
- `成年人上班，先保心情再保全勤`

## Weak To Strong

Weak: `上班真的很累`
Strong: `上班最累的不是干活，是假装这点钱值得`
More resonant: `有时候不是工作太重，是我太容易把它放在心上`

Weak: `同事关系不用太认真`
Strong: `同事只是共事，不是你人生的售后`
More resonant: `同事聊得来是运气，聊不来也不用勉强`

Weak: `不要太在意领导的话`
Strong: `领导一句话，不配租用你一整天的心情`
More resonant: `领导的话可以听，但别让它住进你下班后的脑子里`

Weak: `下班后不要回消息`
Strong: `下班后的我，不提供免费在线客服`
More resonant: `下班后慢一点回，也是在把自己还给自己`

Weak: `开会很浪费时间`
Strong: `有些会开完，唯一产出是大家都老了一点`
More resonant: `有些会开完没解决问题，但至少提醒我该下班了`
